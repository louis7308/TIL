>## 택스트 관련 태그

```
<hn> 제목 <h1>~<h6>
<p> 텍스트 단락 내용이 길면 알아서 창의 너비에 맞게 자동 바꿈
<br> 줄 바꿈
<blockquote> 내용 인용
<strong> 중요한 텍스트 굵게 표시 화면 낭독기 강조
<b> 중요하지는 않지만 굵게 텍스트
<em> 중요한 텍스트를 기울게 표시 화면 낭독기 강조
<i> 중요하지는 않지만 기울게 표시
<ins> 내용을 편집할 때 추가한 내용 표시
<del> 내용을 편집할 때 삭제한 내용 표시
<sup> 위 첨자 ( 지수 생각하면 편함 )
<sub> 아래 첨자
```
>## 목록 관련태그

```
<ol> 순서 있는 목록의 시작과 끝을 나타냄
<ul> 순서 없는 목록의 시작과 끝을 나타냄
<li> 순서 있는 목록이나 순서 없는 목록의 각 항목을 나타냄
<dl> 설명 목록의 시작과 끝
<dt> 설명 목록의 이름을 나타냄
<dd> 설명 목록의 값을 나타냄
```

## 멀티미디어 관련 태그

```
<object> 멀티미디어, PDF 파일 등 다양한 형식 파일 삽입
<embed> <audio>, <video> 태그와 <object> 태그도 지원하지 않을 경우 멀티미디어 파일을 삽입
<img> 이미지 파일을 삽입
<audio> 오디오 파일을 삽입
<video> 비디오 파일을 삽입
